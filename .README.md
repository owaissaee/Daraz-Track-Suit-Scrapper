# Daraz Track Suit Scraper

This project scrapes track suit product data from Daraz.pk using Selenium and BeautifulSoup.

## Features
- Scrapes product name
- Current price
- Original price
- Discount
- Rating
- Sold units
- Location
- Product URL

## Output
Data is saved to `track_suits.csv`.

pip install -r requirements.txt
